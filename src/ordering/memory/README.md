# Memory notes

As you develop your ordering, keep notes here: approaches that worked, dead
ends, and the reasoning behind important choices. One Markdown file per idea or
session is a good rhythm; a `literature/` subfolder (one note per paper, with a
full citation and the idea in your own words) helps when you work from the
research.

This is a convention, not a requirement — nothing here is read by the harness
or the grader. It is purely your own working memory, and it is the first thing
to read at the start of a new session.
